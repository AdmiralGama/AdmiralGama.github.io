Android Apps

Othello

I recreated the game of Othello as a mobile application using Android Studio. It displays a game board with all currently placed pieces, and shows available places where the current player can place a piece. This required the use of advanced logic to calculate available placement positions, as the game of Othello determines where you can place pieces depending on those already placed. Thus, it must take into account every piece already on the board. It also displays whose turn it is, as well as the running score and gives an option to restart the game.

Astronomy Picture of the Day

I created another mobile application using Android Studio that displays an astronomy picture of the day along with its description. This required learning how to access and process results from a REST API, as well as displaying images. I also had to implement error handling for the API, as it does not always succeed in its request and there are multiple possible error states it can return.
