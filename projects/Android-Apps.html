Othello

I recreated the game of Othello as a mobile application using Android Studio. It displays a game board with all currently placed pieces, and shows available places where the current player can place a piece. This required the use of advanced logic to calculate available placement positions, as the game of Othello determines where you can place pieces depending on those already placed. Thus, it must take into account every piece already on the board. It also displays who's turn it is, as well as the running score and gives an option to restart the game.
