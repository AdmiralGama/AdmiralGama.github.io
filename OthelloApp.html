Othello Android App

An app to play the game Othello, with advanced back-end logic to control piece placement and win conditions.

Working in Android Studio, I created an Othello app. It's built with a modular interface system and a seamlessly integrated back-end logic system to allow easy changes and customization. It runs complex logic in the background to compute where the player is allowed to place pieces, and displays the locations on the game board. It's designed to be simple but fast, calculating positions and win conditions as quickly as possible to provide a seamless user experience. This gave me an in-depth understanding of how to build mobile applications, and helped me learn how to make modular applications where components can easily be swapped out and customized to fit the project needs.
